## 🔬 诊断预测模块

### 输入参数
- **人口统计学**: 年龄、性别、ICU类型
- **生命体征**: 心率、血压、MAP、呼吸频率、血氧、体温
- **实验室指标**: 肌酐、BUN、WBC、乳酸、血糖、血小板

### 输出结果
- 表型分类（低危稳定型/进展风险型/高危危重型/恢复转归型）
- 置信度评分
- 预估死亡率
- 诊断依据（关键异常指标）
- 临床建议
- 相似病例数

## 🤖 AI智能分析模块

### 模型性能对比
| 方法 | 轮廓系数 | 死亡率跨度 | 跨中心稳定性 |
|------|----------|-----------|-------------|
| PCA Baseline | 0.0613 | 29.2% | 0.027 |
| S1 Masked | 0.0866 | 17.6% | 0.024 |
| **S1.5 Contrastive** ✓ | **0.0798** | **24.6%** | **0.016** |

### 特征重要性 Top 5
1. 🧪 **乳酸** (18.5%) - 组织灌注指标
2. 🩸 **平均动脉压** (16.2%) - 循环功能
3. ❤️ **心率** (14.8%) - 心脏负荷
4. 🫁 **血氧饱和度** (13.5%) - 氧合功能
5. 🫘 **肌酐** (12.8%) - 肾功能

### 模型架构
- **类型**: Transformer + Contrastive Learning
- **输入**: 21维连续变量 + 观测掩码，48小时时序
- **输出**: 128维患者嵌入向量
- **训练数据**: 7,989例训练，1,141例验证，2,574例测试

## 🔌 API 列表

### 患者管理
```
GET  /api/patients              # 患者列表
GET  /api/patients/<id>         # 患者详情
POST /api/diagnose              # 单患者诊断
POST /api/diagnose/batch        # 批量诊断
GET  /api/diagnose/features     # 诊断字段定义
```

### 数据分析
```
GET  /api/dashboard/stats       # 科研统计数据
GET  /api/ai/analysis           # AI模型分析
```

### 系统配置
```
GET  /api/config/system         # 系统配置
GET  /api/config/ai             # AI模型配置
GET  /api/filters/options       # 筛选选项
```
