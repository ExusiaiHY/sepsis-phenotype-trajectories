# 科研数据看板 - 数据源接入清单

## 已接入数据源

### 1. 基础患者数据 (data/s0/static.csv)
- **患者总数**: 11,986人
- **字段**: 年龄、性别、ICU类型、ICU停留时间、院内死亡率等
- **用途**: 人口统计、ICU分布、死亡率分析

### 2. 时序表型数据 (data/s2/window_labels.npy)
- **维度**: (11986, 5) - 患者 × 5个时间窗口
- **表型类别**: P0(低危)、P1(中高危)、P2(高危)、P3(低中危)
- **用途**: 表型分布、轨迹模式分析

### 3. 轨迹转换统计 (data/s2/trajectory_stats.json)
- **转换事件总数**: 47,944次
- **自我转换**: 42,957次 (89.6%)
- **非自我转换**: 4,987次 (10.4%)
- **转换熵比率**: 0.637
- **用途**: Top转换路径、稳定性分析

### 4. 转换概率矩阵 (data/s2/transition_matrix.json)
- **矩阵维度**: 4×4 (表型间转换)
- **内容**: 各表型间转换次数和概率
- **用途**: 转换流向可视化

### 5. 模型比较数据 (data/s15/comparison_report.json)
- **对比方法**: PCA Baseline vs S1 Masked vs S1.5 Contrastive
- **评价指标**: 
  - Silhouette Score (轮廓系数)
  - Mortality Range (死亡率跨度)
  - Center Distance L1 (跨中心稳定性)
- **结论**: S1.5 Contrastive最优

### 6. 特征定义数据 (data/s0/feature_dict.json)
- **生命体征**: 8维度 (HR/BP/MAP/RR/SpO2/Temp/GCS)
- **实验室指标**: 9维度 (肌酐/BUN/血糖/WBC/血小板等)
- **血气分析**: 4维度 (PaO2/FiO2/PaCO2/pH)
- **总维度**: 21连续变量

### 7. 时序元数据 (data/s2/rolling_meta.json)
- **窗口设置**: 24小时窗口，6小时步长，5个窗口
- **观测密度**: 
  - W0: 27.9%
  - W4: 25.4%
  - **趋势**: 下降 (随时间观测减少)

### 8. 跨中心验证数据 (data/s3/cross_center_report.json)
- **Center A**: 7,989患者 (训练集)
- **Center B**: 3,997患者 (验证集)
- **验证结果**: 
  - 死亡率排序一致: [P0, P3, P1, P2]
  - 稳定率相近: 65.0% vs 64.4%
  - **结论**: 跨中心验证通过

## 数据流转架构

```
原始数据 (data/)
    │
    ├── static.csv ─────────────┐
    ├── window_labels.npy ──────┤
    ├── trajectory_stats.json ──┤──→ Flask后端 (app.py)
    ├── comparison_report.json ─┤      (数据封装+脱敏+计算)
    └── feature_dict.json ──────┘
                                      │
                                      ▼
                              API /api/dashboard/stats
                                      │
                                      ▼
                              前端渲染 (index.html)
                                      │
                    ┌─────────────────┼─────────────────┐
                    ▼                 ▼                 ▼
              核心指标卡片       可视化图表       详细数据表
```
