\section{Software Objectives}
%% ====================================================================

This software implements a three-stage computational framework for self-supervised temporal phenotype trajectory analysis of ICU sepsis patients. The specific objectives include:

\begin{enumerate}[nosep]
  \item Performing standardized preprocessing on multi-center ICU time-series data (PhysioNet 2012, 11,986 patients), including structured missingness handling, outlier clipping, and Z-score normalization
  \item Training a Transformer-based self-supervised encoder via masked value prediction and temporal contrastive learning (Stage 2)
  \item Extracting rolling-window embeddings and performing descriptive temporal phenotype trajectory analysis via per-window clustering (Stage 3)
  \item Building a leakage-aware OOF stacking mortality classifier with probability calibration optimization
  \item Providing comprehensive evaluation covering clustering quality, mortality stratification, cross-center validation, calibration analysis, and bootstrap confidence intervals
  \item Supporting frozen-transfer temporal analysis on external databases (MIMIC-IV, eICU-CRD)
\end{enumerate}

\section{Strengths and Limitations}
%% ====================================================================

\subsection{Strengths}

\begin{enumerate}[nosep]
  \item \textbf{Three-Stage Progressive Architecture}: Static baseline $\to$ self-supervised representations $\to$ temporal trajectory analysis, each stage building on the previous
  \item \textbf{Mask-Aware Self-Supervised Learning}: Explicit observation masks prevent conflating missing values with normality; contrastive window alignment enables rolling-window reuse
  \item \textbf{Comprehensive Calibration Framework}: Five post-hoc methods + structural calibration-aware stacking with 30-config hyperparameter search; final ECE=0.023
  \item \textbf{Clinically Meaningful Outputs}: Probability predictions aligned with true mortality rates (14.1\% vs 14.6\%), suitable for bedside risk communication
  \item \textbf{Cross-Center Validation}: Identical phenotype mortality ordering between training and held-out centers (within PhysioNet 2012 cohort)
  \item \textbf{External Transfer Capability}: Frozen S1.5 encoder processes MIMIC-IV (94K) and eICU (201K) stays with meaningful trajectory recovery
  \item \textbf{Leakage-Aware Evaluation}: OOF stacking with bootstrap CIs, permutation importance, and stratified calibration analysis
  \item \textbf{Modular, Reproducible Design}: YAML-configured, script-driven pipeline with deterministic seeds; each stage independently runnable
\end{enumerate}

\subsection{Limitations}

\begin{enumerate}[nosep]
  \item \textbf{Descriptive Temporal Analysis}: Per-window clustering rather than latent-state inference (no transition probability modeling)
  \item \textbf{Single-Dataset Encoder Training}: S1.5 encoder trained only on PhysioNet 2012; external transfers use frozen weights under incomplete channel overlap
  \item \textbf{Missing Treatment Data}: PhysioNet 2012 lacks true treatment records; proxy indicators are physiological thresholds, not interventions
  \item \textbf{Observational Associations}: Lower mortality in single-transition patients is descriptive, not causal
  \item \textbf{Window Overlap Sensitivity}: 75\% overlap at stride=6h smooths transitions; sensitivity analysis confirms robustness but exact rates depend on stride
  \item \textbf{Data Sparsity Ceiling}: 73.3\% overall missingness, with key biomarkers (bilirubin 98.3\%, lactate 95.9\%) rarely observed
\end{enumerate}

\section{Future Improvement Directions}
%% ====================================================================

\begin{enumerate}[nosep]
  \item \textbf{Treatment-Aware Temporal Phenotyping}: Extend to MIMIC-IV / eICU with true intervention timestamps for phenotype-treatment interaction analysis
  \item \textbf{Source-Specific Encoder Retraining}: Compare frozen transfer vs multi-source pretraining to quantify channel mismatch effects
  \item \textbf{State-Space Modeling}: Replace per-window clustering with HMMs or switching state-space models for probabilistic transition modeling
  \item \textbf{Online Prediction}: Deploy calibrated model for real-time 6h/12h window mortality risk updates at bedside
  \item \textbf{Richer Phenotype Interpretation}: Post-hoc attribution (SHAP), multimodal augmentation, and linkage to organ-support variables
  \item \textbf{Prospective Validation}: Test temporal phenotype trajectories in a prospective clinical study
\end{enumerate}
