\begin{abstract}
\textbf{Background.} Sepsis exhibits substantial clinical heterogeneity, yet most phenotyping approaches rely on static summary features that compress temporal dynamics into single-timepoint representations. Whether learned temporal representations can reveal clinically meaningful phenotype trajectories within the first 48 ICU hours remains underexplored.

\textbf{Methods.} We developed a three-stage computational framework applied to 11,986 ICU patients from the PhysioNet 2012 multi-center database (4~hospitals, 2~centers). First, we established a static phenotyping baseline using statistical features and PCA. Second, we trained a Transformer-based encoder via self-supervised learning (masked value prediction combined with a temporal contrastive window objective) and compared four representation methods across clustering quality, mortality stratification, center stability, and missingness robustness. Third, using the selected self-supervised representation, we extracted rolling-window embeddings (24-hour windows, 6-hour stride) and performed descriptive temporal phenotype trajectory analysis via K-Means clustering on per-window embeddings. All mortality outcomes were obtained from verified PhysioNet Outcomes files (in-hospital mortality rate: 14.2\%).

\textbf{Results.} Static $K{=}4$ clustering on PCA-reduced features identified phenotypes with mortality rates ranging from 6.8\% to 36.0\%. The self-supervised encoder achieved superior center stability (cluster distribution L1 distance = 0.016 vs 0.027 for PCA) and reduced missingness sensitivity ($|r| = 0.15$ vs $0.23$), while PCA retained stronger static mortality separation. Temporal trajectory analysis revealed that 64.8\% of patients maintained a stable phenotype across all windows, while 35.2\% exhibited at least one transition. Stable phenotypes showed strongly stratified mortality (4.0\%, 9.7\%, 22.5\%, 31.7\%; range 27.7 percentage points). Sensitivity analysis with reduced overlap (stride=12\,h) confirmed identical mortality ordering and range (28.0\,pp). Cross-center temporal validation within the PhysioNet 2012 cohort (train on Center~A, evaluate on Center~B) confirmed identical mortality ordering, the same highest-risk phenotype, and a 25.8 percentage-point mortality range on the held-out center.

\textbf{Conclusions.} Self-supervised temporal representations enable descriptive phenotype trajectory analysis that reveals clinically meaningful within-stay dynamics beyond static clustering. Cross-center temporal validation within the PhysioNet 2012 multi-center cohort confirms that phenotype structure transfers across centers, though both centers derive from the same source database. These findings support further investigation with treatment-aware datasets, external database validation, and prospective study designs.

\medskip\noindent
\textbf{Keywords}: sepsis phenotyping, self-supervised learning, temporal trajectories, ICU time-series, multi-center, representation learning
\end{abstract}

\section{Introduction}

\subsection{Background}

Sepsis affects approximately 49 million individuals worldwide annually and is associated with mortality rates ranging from 20\% to over 50\% in intensive care units (ICUs)~\cite{rudd2020}. Despite decades of clinical research, sepsis remains one of the most challenging conditions in critical care medicine. A key contributor to this challenge is the recognition that sepsis is not a single homogeneous disease but rather a syndrome encompassing multiple distinct biological phenotypes~\cite{singer2016}.

The landmark study by Seymour et al.\ (2019) identified four sepsis phenotypes---designated $\alpha$, $\beta$, $\gamma$, and $\delta$---with markedly different clinical trajectories, biomarker profiles, and treatment responses~\cite{seymour2019}. More recently, Antcliffe et al.\ (2025) argued that the true value of subphenotyping lies in predicting heterogeneous treatment effects (theragnostics) rather than prognosis alone~\cite{antcliffe2025}. Feng et al.\ (2025) demonstrated that organ interaction trajectories can be modeled via deep temporal graph clustering with external validation~\cite{feng2025}.

\subsection{Problem Statement}

Existing approaches to sepsis subtyping face several challenges:
\begin{enumerate}[nosep]
  \item \textbf{Static representations}: Most methods compress 48-hour trajectories into summary statistics, discarding temporal dynamics that may distinguish improving from deteriorating patients.
  \item \textbf{Data heterogeneity}: Different ICU databases use different variable nomenclature, measurement frequencies, and data formats.
  \item \textbf{High missingness}: Clinical time-series data are inherently sparse, with laboratory values often measured only once or twice daily.
  \item \textbf{Cross-center generalizability}: Subtypes discovered in a single center may not transfer to other hospitals.
  \item \textbf{Outcome label quality}: Many studies rely on proxy mortality labels rather than verified discharge outcomes.
\end{enumerate}

\subsection{Contributions}

This work makes the following contributions:
\begin{itemize}[nosep]
  \item A \textbf{three-stage phenotyping framework} progressing from static clustering to self-supervised representation learning to descriptive temporal phenotype trajectory analysis.
  \item A \textbf{self-supervised Transformer encoder} trained via masked value prediction and temporal contrastive learning, with systematic comparison against PCA baselines.
  \item \textbf{Descriptive temporal phenotype trajectories} showing that 35.2\% of ICU sepsis patients undergo phenotype transitions within 48~hours, with transitions predominantly toward lower-acuity states.
  \item A \textbf{validated data pipeline} using ground-truth in-hospital mortality outcomes, replacing previously used proxy labels.
  \item \textbf{Robustness validation} including stride sensitivity analysis and center probe analysis confirming no center information leakage.
  \item \textbf{Cross-center temporal validation} within the PhysioNet 2012 multi-center cohort, confirming that phenotype structure and mortality stratification transfer from the training center to a held-out center.
\end{itemize}

\section{Related Work}

\subsection{Sepsis Phenotyping}

The concept of sepsis heterogeneity was first systematically investigated by Seymour et al.\ (2019), who applied latent class analysis to 20,189 sepsis patients~\cite{seymour2019}. Their four-phenotype model has been replicated by subsequent studies~\cite{shankar2020,davido2021}. More recently, deep temporal clustering~\cite{siirtola2022}, contrastive learning~\cite{wang2023}, and organ interaction trajectory modeling~\cite{feng2025} have been proposed. Zhang et al.\ (2024) demonstrated time-series k-means with DTW for sepsis subphenotyping with external validation~\cite{zhang2024dtw}.

\subsection{Self-Supervised Representation Learning for Clinical Data}

Self-supervised representation learning (SSRL) has emerged as a powerful approach for extracting patient representations from unlabeled clinical data. Zheng et al.\ (2025) reviewed 46 SSRL studies, identifying Transformer-based (43\%), autoencoder-based (28\%), and graph-based (17\%) model families~\cite{zheng2025}. Amirahmadi et al.\ (2025) introduced trajectory-ordered objectives that improve downstream prediction beyond standard masked modeling~\cite{amirahmadi2025}.

\subsection{ICU Time-Series Analysis}

ICU data present unique analytical challenges. The PhysioNet/CinC Challenge series has driven methodological advances~\cite{silva2012,reyna2019}. Huang et al.\ (2025) introduced MIMIC-Sepsis, a curated benchmark incorporating treatment variables that substantially improve model performance~\cite{huang2025}.

\section{Methods}

\subsection{System Architecture}

The system implements a three-stage framework (Figure~\ref{fig:pipeline}):

\begin{lstlisting}
Stage 1: Data -> Preprocessing -> Features -> PCA -> K-Means
Stage 2: Time Series + Masks -> Transformer -> Self-Supervised Pretraining -> Embeddings
Stage 3: Rolling Windows -> Per-Window Embedding -> Temporal Clustering -> Transitions
\end{lstlisting}

Each stage builds on the previous, with modules designed as independent components enabling systematic comparison across representation methods.

\begin{figure}[H]
\centering
\includegraphics[width=0.95\textwidth]{pipeline_diagram.png}
\caption{Three-stage framework for temporal sepsis phenotyping. Stage~1 establishes a static PCA baseline. Stage~2 trains a self-supervised Transformer encoder via masked prediction and contrastive learning. Stage~3 extracts rolling-window embeddings for descriptive temporal phenotype trajectory analysis.}
\label{fig:pipeline}
\end{figure}

\subsection{Data Sources and Outcome Labels}

\subsubsection{PhysioNet 2012 Multi-parameter ICU Database}

The primary dataset contains records from 12,000 ICU patients across four hospitals~\cite{silva2012}. After quality filtering, 11,986 patients were retained:
\begin{itemize}[nosep]
  \item \textbf{Center~A} (set-a + set-b): 7,989 patients
  \item \textbf{Center~B} (set-c): 3,997 patients from a separate hospital within the PhysioNet 2012 cohort
\end{itemize}

\textbf{Outcome labels.} Ground-truth in-hospital mortality was obtained from PhysioNet Outcomes files for all 11,986 patients (mortality rate: 14.2\%). An earlier version of this analysis used proxy mortality derived from physiological thresholds (GCS~$\leq$~5 or sustained MAP~$<$~55\,mmHg). Outcome audit revealed that this proxy had a positive predictive value of only 17.7\%. All results in this paper use verified outcome labels.

\textbf{Data schema.} Following CLIF-inspired design principles~\cite{rojas2025}, the data layer separates continuous measurements (21~variables), observation masks, and proxy-derived indicators into distinct tensors. Proxy indicators (MAP~$<$~65, FiO$_2$~$>$~0.21) are explicitly labeled as proxies and stored separately from true treatment records, which are unavailable in PhysioNet~2012.

\subsubsection{Supplementary Datasets}

A configurable synthetic data generator with known ground-truth labels enables clustering recovery evaluation (ARI, NMI). A MIMIC-IV mock database~\cite{johnson2023} validates pipeline compatibility.

\subsection{Preprocessing}

All measurements are resampled to a uniform hourly grid over 48~hours using LOCF. Outliers exceeding $4\sigma$ are clipped. Missing values are imputed via forward fill (up to 6~hours), global median, and zero-fill as fallback. Observation masks recording which values were truly measured versus imputed are preserved as first-class inputs. Overall missing rate is 73.3\%.

\subsection{Static Feature Engineering (Stage~1)}

From the preprocessed 3D tensor ($n \times 48 \times 21$), we extract a 2D feature matrix using six statistical functions (mean, std, min, max, trend, last value) across three windows (12\,h, 24\,h, 48\,h), yielding 378 features plus derived clinical indicators. PCA reduces this to 32~dimensions (62.5\% variance explained).

\subsection{Self-Supervised Representation Learning (Stage~2)}

\subsubsection{Encoder Architecture}

A Transformer-based encoder processes the hourly time series with observation masks as explicit input channels. The input is formed by concatenating value and mask vectors: $\mathbf{x}_\text{input} = [\mathbf{x}; \mathbf{m}] \in \mathbb{R}^{T \times 42}$. A linear projection maps this to $d_\text{model} = 128$ dimensions. Learnable positional encodings provide temporal position awareness. A 2-layer, 4-head Transformer encoder with pre-layer normalization and GELU activation produces per-timestep representations, aggregated via observation-density-weighted mean pooling.

\subsubsection{Pretraining Objectives}

\textbf{Masked value prediction.} 15\% of observed values are randomly masked. The encoder must predict original values at masked positions. Loss: MSE on masked observed positions only.

\textbf{Temporal contrastive window objective.} Two stochastic 30-hour views are extracted with constrained overlap (12--24~hours). Both are encoded by the shared encoder, then projected through a 2-layer MLP projection head ($128 \to 128 \to 64$). NT-Xent loss ($\tau = 0.1$) with in-batch negatives. The projection head is discarded after pretraining.

\textbf{Combined loss:} $\mathcal{L} = \mathcal{L}_\text{masked} + \lambda(t) \cdot \mathcal{L}_\text{contrastive}$, where $\lambda$ warms up linearly from 0 to 0.5 over 10~epochs.

\subsubsection{Representation Comparison}

Four representations are compared: (1)~PCA baseline (32d), (2)~masked reconstruction only (S1, 128d), (3)~masked + contrastive (S1.5, $\lambda{=}0.5$, 128d), and (4)~reduced contrastive ablation (S1.6, $\lambda{=}0.2$, 128d).

\subsection{Descriptive Temporal Phenotype Trajectories (Stage~3)}

Using the frozen S1.5 encoder, rolling-window embeddings are extracted: 5~windows of 24~hours with 6-hour stride. K-Means ($K{=}4$) is fit on train-split windows only (30,955 windows) and applied to all 59,930 windows. Patients are classified as \emph{stable} (all 5 labels identical), \emph{single-transition} (1~change), or \emph{multi-transition} ($\geq$2 changes). All temporal analyses are descriptive; no latent-state models or causal claims are involved.

\subsection{Evaluation and Robustness}

\textbf{Clustering:} Silhouette score, mortality separation. \textbf{Probes:} Mortality AUROC, center AUROC ($\downarrow$ better), ICU LOS R$^2$, density-norm correlation. \textbf{Multi-seed:} 5~random seeds for all KMeans. \textbf{Stride sensitivity:} Repeated with stride=12\,h (50\% overlap). \textbf{Implementation:} Clustering and linear probes used scikit-learn~\cite{pedregosa2011}; figures were generated with Matplotlib~\cite{hunter2007}.

\section{Results}

\subsection{Static Phenotyping Baseline (Stage~1)}

After quality filtering, 11,986 patients were retained (Center~A: 7,989; Center~B: 3,997). Ground-truth in-hospital mortality was 14.2\%. Static $K{=}4$ clustering on PCA-reduced features identified phenotypes with mortality rates of 6.8\%, 10.0\%, 33.8\%, and 36.0\% (range 29.2 percentage points). Silhouette score was 0.061.

\subsection{Descriptive Temporal Phenotype Trajectories (Stage~3)}

\subsubsection{Temporal Stability and Transitions}

Of 11,986 patients, 7,764 (64.8\%) maintained a stable phenotype across all five windows, 3,509 (29.3\%) exhibited one transition, and 713 (5.9\%) exhibited multiple transitions.

The 27.7 percentage-point range across stable temporal phenotypes exceeded the static S1.5 result (24.6\%), suggesting that temporally stable patients form purer phenotype subgroups.

\subsubsection{Transition Pathways and Mortality}

The most frequent non-self transitions were Phenotype~$1 \to 0$ (956~events), $1 \to 3$ (728), and $3 \to 0$ (713), collectively representing movement toward lower-acuity phenotypes. Single-transition patients had lower mortality (11.4\%) than stable patients (15.4\%) or multi-transition patients (15.2\%). This descriptive association suggests that phenotype transitions---particularly from higher-risk to lower-risk states---may reflect favorable clinical trajectory change, though causal interpretation requires controlled study designs.

\subsubsection{Temporal Prevalence Shift}

Phenotype~0 (lowest mortality) increased in prevalence from 24.6\% at window $[0, 24)$ to 33.1\% at window $[24, 48)$, while Phenotype~1 decreased from 35.7\% to 28.1\%. This population-level shift is consistent with clinical stabilization during the first 48~ICU hours, though it may also reflect survivor bias or measurement pattern changes.

\section{Discussion}

\subsection{Three-Level Phenotyping Framework}

This work demonstrates a progression from static clustering through self-supervised representation learning to descriptive temporal phenotype trajectory analysis. Each stage addresses a limitation of the previous: Stage~2 replaces hand-crafted features with learned representations robust to missingness; Stage~3 reveals within-stay dynamics that static methods cannot capture.

The temporal analysis shows that approximately one-third of patients undergo phenotype transitions within 48~hours. The clinical significance is supported by three findings: (1)~stable phenotypes show strongly stratified mortality (4.0\%--31.7\%), (2)~the most common transitions move toward lower-risk phenotypes, and (3)~single-transition patients have lower mortality than stable patients, consistent with favorable clinical trajectory change.

\subsection{Representation Learning Trade-offs}

The self-supervised encoder (S1.5) offered improved center stability and reduced missingness sensitivity compared to PCA, though PCA retained stronger static mortality separation. This trade-off is characteristic of learned representations: they capture more generalizable patterns at the cost of discriminative specificity. The selection of S1.5 was motivated primarily by its ability to process variable-length sub-sequences directly, which PCA lacks.

\subsection{Limitations}

\begin{enumerate}[nosep]
  \item \textbf{Descriptive, not model-based}: The temporal trajectories arise from per-window clustering, not latent-state inference. This limits probabilistic transition modeling.
  \item \textbf{Overlap sensitivity}: Window overlap (75\%) smooths embedding changes. Sensitivity analysis with 50\% overlap confirmed mortality stratification but showed higher transition rates, suggesting the primary analysis is conservative.
  \item \textbf{Missing treatment data}: PhysioNet 2012 lacks true treatment records. Treatment-aware phenotyping requires datasets with documented interventions~\cite{huang2025}.
  \item \textbf{Observational associations}: The lower mortality in transitioning patients is a descriptive association, not a causal effect.
  \item \textbf{Single-dataset scope}: Cross-center temporal validation within the PhysioNet 2012 cohort confirms that phenotypes transfer across centers (identical mortality ordering, 25.8\,pp range on held-out Center~B), but both centers derive from the same source database. External validation on independently collected cohorts (e.g., eICU-CRD~\cite{pollard2018}) is needed.
  \item \textbf{Data sparsity}: The 73.3\% missing rate limits signal for both representation learning and clustering.
\end{enumerate}

\subsection{Future Directions}

\begin{enumerate}[nosep]
  \item Treatment-aware temporal phenotyping using MIMIC-IV or eICU with documented interventions.
  \item Latent-state models (HMMs, switching state-space models) for probabilistic transition modeling.
  \item External database validation: validating temporal phenotypes on independently collected ICU databases (eICU-CRD~\cite{pollard2018}, MIMIC-IV) to establish generalizability beyond the PhysioNet 2012 cohort.
  \item Bedside real-time phenotype assignment from early (6\,h, 12\,h) data windows.
\end{enumerate}

% ============================================================
\section{Conclusion}

This study presents a three-stage framework for sepsis phenotyping that progresses from static clustering to self-supervised representation learning to descriptive temporal phenotype trajectory analysis. Applied to 11,986 multi-center ICU patients with ground-truth mortality outcomes, the framework identifies four phenotypes with strongly stratified mortality (4.0\%--31.7\% for temporally stable patients). Temporal analysis reveals that 35.2\% of patients undergo phenotype transitions within 48~hours, with transitions predominantly toward lower-acuity phenotypes and conservative transition rate estimates confirmed by stride sensitivity analysis. Cross-center temporal validation within the PhysioNet 2012 cohort confirms that phenotype structure, mortality ordering, and transition patterns transfer from the training center to a held-out center, though both centers derive from the same source database. These descriptive findings demonstrate that temporal phenotype trajectories provide clinically meaningful and reproducible information beyond static subtyping, supporting further investigation with treatment-aware datasets, external database validation, and prospective study designs.
