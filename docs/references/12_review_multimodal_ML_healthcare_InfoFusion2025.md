# Review of Multimodal Machine Learning Approaches in Healthcare

**Source**: Information Fusion (ScienceDirect, February 2025)
**URL**: https://www.sciencedirect.com/science/article/pii/S1566253524004688
**Access**: Paywalled (Elsevier) - summary from abstract and search results
**Relevance**: Comprehensive review of multimodal ML in healthcare with dataset catalog

---

## Overview

Provides an overview of multimodal machine learning approaches in healthcare, encompassing data modalities commonly used in clinical diagnoses: imaging, text, time series, and tabular data.

## Key Contributions
- Overview of **17 multimodal clinical datasets**
- Review of **50+ studies** on multimodal integration
- Predominant focus on imaging + tabular data integration

## Modalities Covered
- Medical imaging (radiology, pathology)
- Clinical text (notes, reports)
- Time series (vitals, waveforms)
- Tabular data (demographics, labs, scores)

## Key Findings
- Imaging + tabular data is the most studied modality combination
- Recent advances in generative learning enhance fusion capability
- Standardized evaluation protocols are still lacking
- Missing modality handling remains an open challenge

## Key Takeaways for Our Project
- 17 multimodal clinical datasets cataloged (reference for dataset selection)
- Imaging + tabular is well-studied; EHR time series + notes is less explored (opportunity)
- Need standardized evaluation to enable cross-study comparison
